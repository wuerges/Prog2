Trabalhos

DescartesLab 
              O DescartesLab foi utilizado como projeto na Olimpíada do Conhecimento 2016, tendo em vista o descarte incorreto de resíduos,
          ele trabalha fazendo uma interface entre as empresas de reciclagem e o usuário final. Desenvolvido não só por Gabriel, mas também
          por Andrew Malta e Diovanna Caroline Schell, o projeto recebeu Medalha de Prata ficandoem 2ndo lugar na seletiva nacional.
          Visite o projeto clicando aqui.
Neon Shades 
             Neon Shades foi um projeto desenvolvido em conjunto com Thiago Gugel para a disciplina de Algoritmos e Programação, como
          requisito parcial para aprovação na matéria. Consistia de um jogo baseado no mundialmente conhecido Doodle Jump, no qual o
          usuário é representado por um personagem e precisa escalar a tela para obter pontos.
Nox Games 
             Desenvolvido em conjunto com Solano Santos de Lima, este projeto concorreu à Situação de Aprendizagem SC/2016. Trata-se de um
          website voltado ao público infantil para aprendizado. Contando com diversos quizes interativos, este projeto ganhou o 3o lugar
          na competição estadual entre 17 concorrentes.
